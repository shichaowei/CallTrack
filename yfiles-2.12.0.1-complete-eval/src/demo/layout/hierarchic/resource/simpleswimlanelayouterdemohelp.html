Description

This simple demo shows how to use the swim lane feature of the new IncrementalHierarchicLayouter.

It can either calculate a new layout or calculate a layout taking the current arrangement into account or place selected nodes incrementally in the current graph.

Usage

Adding swimlanes or moving nodes.
Left-click on the graph to create a new node. Its label and thus its assigned swimlane is automatically set to the new number of existing nodes.
To change the swimlane of specific nodes, select one or more nodes, right-click one selected node to open a context menu, and choose Move to lane and the number of the desired lane.
Layout
Calculates a new layoutfor all elements in the graph.
The swimlane assignment determines the horizontal position of nodes. The graph structure determines the vertical position of nodes.
Layout From Sketch
Calculates a new layout while taking the current arrangement into account.
The swimlane assignment determines the horizontal position of nodes. The current vertical position determines the calculated vertical position of nodes (instead of the graph structure).
Layout Incrementally
Arranges the selected nodes while keeping the relativei> positions of all other elements in the graph as is.
