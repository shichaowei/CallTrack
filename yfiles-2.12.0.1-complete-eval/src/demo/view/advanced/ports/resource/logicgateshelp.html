Feature description

Changing the Diagram

Adding a New Logic Gate
Clicking on an empty spot in the main panel adds a new logic gate to the diagram. It is of the kind which is currently selected in the list on the left. Gates can also be added using drag and drop.
Connecting Logic Gates
Possible start pins for new connections are indicated with a green color, if the mouse hovers over them. New connections start from the indicated pin. While creating a connection the possible end pins are indicated in green if the mouse hovers over them.
Reassigning a Connection
The pins of a connection can be reassigned individually using drag and drop. While moving a pin of a connection possible target pins are indicated in green.

Layout

In yFiles for Java Complete it is possible to (re-)layout the complete diagram by pressing the Layout toolbar button. Note that the layout does not change the relative locations of the connections at their start and end gates.
