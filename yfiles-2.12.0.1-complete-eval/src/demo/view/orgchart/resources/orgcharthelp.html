Organization Chart

Mouse Interaction

Mouse Wheel
Changes the zoom level of the view.
Middle Button/Mouse Wheel Drag
While pressed, a local zoom magnifier appears at the current cursor location.
Left Mouse Drag
Move selected node.
Right Mouse Drag
Drag mode to move the diagram within the view.
Click on Node
Selects the node in both the Chart and the Structure View. Shows the corresponding data in the properties view.
Double-Click on Employee
When in Global View mode, zoom in on the employee. When in Local View mode, the clicked employee will become the new central element of the local view.
Double-Click on Business Unit
Toggles collapsed and expanded state of the business unit. In collapsed state the members of a business unit are not shown. To use this feature, Show Business Units needs to be checked.

Key Bindings

To use keyboard navigation the chart needs to have the focus.

Left Arrow Key
Moves the diagram to the right
Right Arrow Key
Moves the diagram to the left
Up Arrow Key
Moves the diagram down
Down Arrow Key
Moves the diagram up
CTRL + Left Arrow Key
Selects a node to the left of the currently selected node
CTRL + Right Arrow Key
Selects a node to the right of the currently selected node
CTRL + Up Arrow Key
Selects a node above the currently selected node
CTRL + Down Arrow Key
Selects a node below the currently selected node
+ Key
Zoom In
- Key
Zoom Out
Enter
When in Global View mode, zoom in on the selected node. When in Local View mode, the selected node will become the new central element of the local view.

Overview

Mouse Interaction

Mouse Drag highlighted region
Move the diagram in the main view
Mouse Drag corner of highlighted region
Change zoom level of the main view

View Option

Global View
Show the complete organization chart.
Local View
Limit view to selected employee and her immediate superiors and subordinates. When Show Colleagues is checked, colleagues of same rank are shown as well.
Show Business Units
If checked, the organization chart will display business units as boxes around the members of each unit. Business Units can be collapsed and expanded by double clicking on them.
