Distributing Applications

Obfuscation

As stated in section 2 of the Software License Agreement, you are required to obfuscate the yFiles Redistributables (i.e., the y.jar file) before distributing your application to end users or to anyone who does not have an appropriate yFiles for Java license.

The minimum requirement of the obfuscation process is that all class, method and field names of the classes contained in the Software Redistributables are obfuscated.

Obfuscation can easily be integrated into your deployment process and can be done using yWorks' free Java byte code obfuscator yGuard.

Appendix A: Obfuscation of the Developer's Guide provides theory and techniques for obfuscating Java code.
