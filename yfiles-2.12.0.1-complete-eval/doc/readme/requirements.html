yFiles for Java Technical Requirements

Prerequisites

Development

To build applications based on yFiles for Java you need to have a Java Software Development Kit version 1.4 or higher installed on your system.

We suggest using Java SE 8 Update 40 JDK or higher from Oracle which is freely downloadable at http://www.oracle.com/technetwork/java/javase/downloads/index.html for Microsoft Windows, Linux, and Solaris operating systems.

Deployment

To run yFiles for Java applications, including the provided demo applications, you will need to have a Java Runtime Environment (JRE) version 1.4 or higher installed. We suggest using Java SE 8 Update 40 JRE from Oracle which is freely downloadable at http://www.oracle.com/technetwork/java/javase/downloads/index.html for Microsoft Windows, Linux, and Solaris operating systems.
