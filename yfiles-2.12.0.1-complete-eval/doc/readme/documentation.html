yFiles for Java Demos and Documentation

A number of resources are included to help you learn more about using yFiles for Java:

Included Resources

Redistributables

The y.jar file in the lib/ directory is the Redistributable as mentioned in Section 2 in the license terms.

Source Code Demos

The src/ directory contains extensive tutorial source code demo applications.

To see the demos in action, simply set up a project in your IDE that uses the src/ directory as the source classpath and the y.jar file as library.

For the latter, you'll want to attach the API documentation to the y.jar file so that your IDE can show documentation tooltips, as described under installation.
IDEA users additionally need to add resource patterns to their compiler settings (also described under installation).

To step through the demos and start them one after the other, we've provided an interactive demo browser: class demo.DemoBrowser in the src/ directory. Compile and run this DemoBrowser class in your IDE.

Alternatively, you can also use the Ant build script that we've included to run the DemoBrowser class. To launch the interactive demo browser, run the Ant target named "run".

Getting Started Tutorial

The tutorial/ directory contains the Getting Started Tutorial, a step-by-step introduction to the yFiles diagramming library with dedicated source code examples. The tutorial also points to related topics in the Developer's Guide and Knowledge Base.

Developer's Guide

The doc/developers-guide/ directory contains the more advanced, comprehensive Developer's Guide which describes the architecture and the main concepts used in yFiles. This HTML version is also available online, or you may download a PDF of the Developer's Guide through your yFiles downloads link.

API Documentation

The doc/api/ directory contains Javadoc™ API documentation of the yFiles for Java library. UML-enhanced PNG and SVG versions are also available online.

You'll want to attach the API documentation to the y.jar file so that your IDE can show documentation tooltips, as described under installation.
