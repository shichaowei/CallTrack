Tutorial Demo Code

The following tutorial demo applications show how to use the TableGroupNodeRealizer for tabular data representation:

  • TableStyleDemo.java shows how to use TableStyle implementations with the predefined TableNodePainter configurations to configure colors and line types.
  • SwimlaneDemo.java demonstrates user interaction and shows how to customize it. The demo also presents a way to create new rows and columns via drag'n'drop gestures.
